# A Test Odyssey: Exploring Intelligent Agents in End-to-End Testing

## Workshop Overview

Step into the future of testing with this hands-on workshop introducing intelligent agents in end-to-end automation using Playwright and its ecosystem. Learn how agents can plan, generate, and heal tests using Playwright MCP (Model Context Protocol) to create more adaptive, resilient, and context-aware automation.

## Exercises Overview

| Exercise | Time | Topic | Key Skills |
|----------|------|-------|------------|

| 1 | 20 min | Introduction to Playwright MCP & Agents | Agent Concepts, MCP Fundamentals |
| 2 | 25 min | Agent-Driven Test Planning | AI-Assisted Planning, Test Strategy |
| 3 | 30 min | Intelligent Test Generation | Auto-Generation, Agent Workflows |

## Common Questions & Issues

### Technical Problems

**Q:** Playwright tests fail to run  
**A:** Check Node.js version, reinstall dependencies, verify application is running

**Q:** Elements not found in tests  
**A:** Use `npx playwright codegen` to generate reliable selectors

**Q:** Tests run too slowly  
**A:** Discuss test parallelisation and performance optimisation

### Conceptual Questions

**Q:** When should I write automated vs manual tests?  
**A:** Discuss cost/benefit, repeatability, and maintenance considerations

**Q:** How much testing is enough?  
**A:** Risk-based testing approach, critical path coverage

**Q:** How do I convince my team to adopt testing?  
**A:** Start small, demonstrate value, measure impact
