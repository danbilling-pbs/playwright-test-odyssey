## Common Questions & Issues

### Technical Problems
**Q:** Playwright tests fail to run  
**A:** Check Node.js version, reinstall dependencies, verify application is running

**Q:** Elements not found in tests  
**A:** Use `npx playwright codegen` to generate reliable selectors

**Q:** Tests run too slowly  
**A:** Discuss test parallelisation and performance optimisation

### Conceptual Questions  
**Q:** When should I write automated vs manual tests?  
**A:** Discuss cost/benefit, repeatability, and maintenance considerations

**Q:** How much testing is enough?  
**A:** Risk-based testing approach, critical path coverage

**Q:** How do I convince my team to adopt testing?  
**A:** Start small, demonstrate value, measure impact
